How I Did It
The gap + 3 engine layers used

What you can't see after accepting an offer

Every existing mode is pre-offer. Mine is the only post-offer mode in the repo.
"They sponsor H-1B" ≠ they win petitions reliably at the right wage level
Wage Level I/II petitions face more USCIS scrutiny — higher RFE and denial risk
SOC code mismatch — employer picks the code, you don't see the LCA until filing
Backup options decay while you're employed — if something goes wrong you start from zero
